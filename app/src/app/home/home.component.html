Angular app

Assignment 2

Features

This is an Assignment done for BTI 425, some features include:

  • View and browse a list of all terms
  • Search for a term
  • View details about a term, and its meaning in English and in other languages
  • Add a new term, English, or in another language (including its first or initial definition)
  • For an existing term (English or other), add another definition/data

Description

the purpose or objective of the assignment is to create a web API and an Angular app that has good coverage of the topics in the course. The app’s purpose is to define or translate the English-language terminology we use in the computer programming courses in the School of SDDS, for use by all students, whether or not English is a student’s first language. We have so many students who are learning English and computer programming at the same time, and that is a challenging combination. The technical terminology is often unclear, complicated, and sometimes isn’t explained well. For example, when we say “asynchronous”, what does it clearly mean in English, as well as in a student’s first language?
